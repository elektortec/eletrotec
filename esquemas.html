Visão Geral do Projeto

📋 Bloco Funcional

Descrição

Oscilador de referência (≈240 Hz) gerado com CD40106 (inversores Schmitt). Saída alimenta um CD4017 que alterna duas saídas para conduzir drivers MOSFET em configuração push-pull sobre o transformador 12V→220V.

⚙️ Aplicação

Uso

Pequena carga resistiva ou iluminação; não recomendado para cargas sensíveis sem filtragem e regulação adicionais.

🔒 Segurança

Atenção

Perigo de choque e incêndio. Trabalhe com isolamento, fusíveis e em baixa potência para testes.

Lista de Componentes

  • IC1: CD40106 (inversor Schmitt) — oscilador ~240 Hz (use 2 estágios em cascata para estabilidade)
  • IC2: CD4017 (decade counter) — divide/seleciona saídas 1 e 6 para alternância
  • MOSFETs: 2x N-channel logic-level (ex.: IRLZ44N ou equivalente de baixa Rds(on))
  • Transformador: 12V CT para 220V (primário 12-0-12, secundário 220V) — ou transformador com primário adequado para push-pull
  • Resistores: 10k pull-downs, 100Ω gate resistors
  • Capacitores: 10nF–100nF para o oscilador; capacitores de desacoplamento 0.1µF
  • Fusível: 10A (ajuste conforme projeto)
  • Fonte: Bateria 12V (capacidade conforme corrente esperada)

Conexões e Diagrama (descrito)

Oscilador (CD40106)

  • Use um estágio do CD40106 como oscilador RC: entrada do inversor em resistor série (p.ex. 100k) e capacitor para GND (p.ex. 10nF–47nF). Ajuste para ~240 Hz.
  • Se necessário, encascadeie dois estágios para formas de onda mais quadradas e maior amplitude de saída.

Contador/Divisor (CD4017)

  • Saída do oscilador vai ao clock do CD4017 (pin CLK).
  • Configure CD4017 para alternar entre duas saídas usadas para conduzir os gates dos MOSFETs: por exemplo, usar Q0 e Q5 (saídas 1 e 6) para obter meia-período alternado; prenda as demais saídas a GND via pull-down se necessário.
  • Conecte RESET do CD4017 a GND para contagem contínua; use ENABLE (se aplicável) conforme datasheet.

Drivers e Estágio de Potência

  • Use resistores de gate (100Ω) entre saídas do CD4017 e gates dos MOSFETs.
  • Coloque pull-downs (10k) dos gates para GND para evitar condução indesejada.
  • Fonte 12V alimenta os drains dos MOSFETs (configuração push-pull): os MOSFETs alternam a ligação das extremidades do primário do transformador ao +12V, criando alternância na secundária em alta tensão.
  • Adicione diodos de flyback se o transformador e o layout exigirem supressão transiente.
